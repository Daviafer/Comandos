-----JAVA-----

Após baixar JDK (salvando em outro disco)

sudo gedit .bashrc

export JAVA_HOME=/media/user/Dados/DaviF/EstudoJAVA/OpenJDK-16
export PATH=$PATH:$JAVA_HOME/bin

FECHAR TERMINAL OU ATUALIZAR COM :

source .bashrc

DICA:
no arquivo ~/.profile ou mesmo ~/.bashrc (vale apenas pro seu usuário), no /etc/profile ou mesmo /etc/bash.bashrc (vale pra todos os usuários).
após editar um desses arquivos você precisa fazer logoff e login novamente ou recarregar o ambiente alterado: source <arquivo alterado>

-----GRADLE-----

Após baixar, descompactar o Gradle (salvando em outro disco)

sudo gedit .bashrc

export GRADLE_HOME=/media/user/Dados/DaviF/EstudoJAVA/gradle-bin-7.0/bin
export PATH=/media/user/Dados/DaviF/EstudoJAVA/gradle-bin-7.0/bin:${PATH}

OUTRA FORMA (mais simples, talvez)

baixar arquivo webste oficial
mkdir /opt/gradle = criando pasta no diretório /opt/
abrir local que está o programa baixado
unzip -d /opt/gradle gradle-7.0-bin.zip (ou a versão baixada)  = extrair e colocar aquivos na pasta criada
ls /opt/gradle/gradle-7.0  = listar aquivos
export PATH=$PATH:/opt/gradle/gradle-7.0/bin = exportando arquivos do /bin pro PATH
